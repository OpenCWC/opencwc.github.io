Getting Started

This page will get you setup with the project files and workflow

Downloading the project files

Every character and environment provided on the OpenCWC website is available in two variations, and the best choice will depend on the software you plan to use. For those that use Autodesk Maya, choose the Maya option. This contains a tidy Maya project directory with scenes, source images and other assets. For everyone else, use the generic option. This contains the geometry and textures exported from Maya and exists for accessibility reasons.

This documentation shows the approriate workflow with Autodesk Maya, RenderMan and After Effects.

Every now and then updates will be released for individual projects, which will be accompanied with a list of modifications and additions.

Downloading Autodesk Maya 2022.4

Oftentimes, projects made in Autodesk Maya will be done entirely in a specific version, as updating the software has the potential to break things. The OpenCWC Project was made with Maya 2022.4, so you'll need this version to use the project.

Installing the OpenColorIO plugin for After Effects

The OpenColorIO plugin for After Effects provides a way to manage ACEScg renders in an identical way to RenderMan's preview interface.

Download the OpenColorIO plugin from the homepage and extract the archive. Move the ACES-1.2, basic and filmic-blender folders to C:\ProgramData\OpenColorIO (create this folder if it doesn't already exist). Move the plugin file to the After Effects plugin folder. A quick Google search will tell you where this is.
